Le projet en bref

Première vraie incursion dans l'administration système : installation d'une machine virtuelle Debian sous VirtualBox, puis configuration d'un serveur LAMP complet — Linux, Apache, MySQL, PHP.

L'objectif : disposer d'un environnement de développement web fonctionnel, comme dans une vraie entreprise.

Déroulement

1

Installation de la VM Debian

Configuration de VirtualBox, installation de Debian, partitionnement du disque, réseau NAT.

2

Configuration LAMP

Installation et configuration d'Apache, MySQL et PHP ; premiers scripts PHP connectés à la base.

3

Déploiement & tests

Déploiement d'une application web simple sur le serveur et vérification du fonctionnement complet.

Ce que j'en retiens

Comprendre comment fonctionne un serveur web de l'intérieur change la façon de développer. Je ne vois plus un site de la même manière : je pense désormais à Apache, aux droits des fichiers, à la configuration PHP. Cette SAÉ m'a donné envie de comprendre l'infrastructure.

Compétences mobilisées

Administrer

Installer et configurer un environnement serveur complet et y déployer une application.

Réaliser

Diagnostiquer et résoudre des problèmes de configuration jusqu'au bon fonctionnement de bout en bout.
