Le projet en bref

À partir d'un jeu de données réel sur les catastrophes climatiques par pays, j'ai mené tout le cycle : analyse des données brutes, modèle conceptuel (MCD), modèle logique (MLD), script SQL de création, import, puis visualisations en Python.

Le projet a combiné compétences en bases de données et en traitement de données avec Python (pandas, matplotlib) pour produire des visualisations pertinentes et lisibles.

Déroulement

1

Analyse & modélisation

Étude du jeu de données climatiques, identification des entités, conception du MCD et du MLD.

2

Implémentation SQL

Création du schéma, import des données, requêtes d'agrégation pour préparer la visualisation.

3

Visualisations Python

Graphiques matplotlib / seaborn : évolution temporelle, comparaisons par région, corrélations.

Ce que j'en retiens

La SAÉ qui m'a donné le goût de la data. Transformer des données brutes en graphiques qui racontent une histoire — les catastrophes climatiques sur 50 ans — c'est à la fois technique et porteur de sens. Python + pandas est devenu un de mes combos préférés.

Compétences mobilisées

Analyser

Explorer un jeu de données réel et en extraire des visualisations qui font sens.

Gérer

Mener le cycle de données complet, de la modélisation à l'import jusqu'à l'exploitation.
